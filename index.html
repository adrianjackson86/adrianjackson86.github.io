Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you travel closer to the equator. To accomplish the analysis, weather date is retrieved using the OpenWeatherMap API to build a dataset for 500 cities.

Matplotlib is used to plot visual results of the weather as it relates to latitue. Weather insights include: humidity, cloudiness, wind speed, and max temperature.
